Настройка обмена данных 1С с ВМС.

Описание.

Инструкция описывает настройки обмена между 1С и ВМС, а именно:

  • Какие контракты на какой склад должны выгружаться.
  • Настройки складов 1С, документы которых должны выгружаться в ВМС.
  • Настройка прав Логиста.
  • Настройка уведомлений для Операторов и Бухгалтеров.

Подробное описание действий:

  • 1Открыть «1С»
  • Открыть «Операции»
  • Открыть «Справочник»
  • В появившемся списке выбрать «EME.WMS Склады»
  • Появится список существующих складов, выбираем нужный и открываем его.

В открывшемся окне можно выполнить следующие действия:

  • Добавить получателей отчёта, содержащего перечень клиентов, для которых оператор в корректировках указал значение срока большее, чем определено договором с этим клиентом.
  • Определить номенклатурные группы, которые будут выгружаться на склад.
  • Необходимо заполнить следующие поля:
    • Номенклатурная группа. Выбирается из справочника.
    • Синоним. Одно слово, которое описывает номенклатурную группу (или несколько). Не должно быть пробелов, точек, запятых и прочих специальных символов. Только буквы и цифры. Номенклатурные группы с одинаковыми синонимами выгружаются как один документ. И собираются сборщиком как один документ.
    • BBD для склада - Минимальный срок годности. Число, означающее минимальный остаток срока годности, при котором товар всё ещё может уехать со склада без заведения промоактивностей. Обычно это 21 день, но для товара, являющегося скоропортом – это значение должно быть меньше.
    • Позаявочно. Определяет, как будут выгружаться заявки. Либо объединенные в одну сводную, либо каждая заявка в отдельности. ВАЖНО! Заявки выгружаются только после их включения в состав сводной накладной.
    • BBD для холодильника. Минимальный остаток срока годности, при котором товар, хранящийся в рефрижераторных контейнерах, может уходить клиентам.
    • Получатель рассылки. Здесь указывается эл. адрес человека, который будет получать письма с результатом «предпросчёта» заявок в 1С, учитывая актуальные остатки в ВМС. При необходимости добавить нескольких людей в рассылку, сделать это можно разграничив их адреса точкой с запятой – « ; ».
  • Определить, документы каких складов должны выгружаться на текущий склад ВМС. Этот пункт настраивается только администратором ВМС при запуске склада.
  • Указать, кто будет логистом на данном складе. Внесение пользователя в этот список даёт ему право выгружать сводные накладные на определённую дату.
  • Указать, операторов и бухгалтеров, которым нужно напоминание о выгрузке таких документов как «Сводная накладная» или «Списание»
